HTML E CSS

JavaScript + TypeScript

Saia do exato zero, e consiga fazer projetos avançados com uma das linguagens mais populares do mercado.

React

O React é uma biblioteca JavaScript de código aberto com foco em criar interfaces de usuário em páginas web.

NodeJS (ExpressJS)

O Node. js é uma plataforma em que é possível criar aplicações Javascript sem depender de um browser para a execução.

Git + GitHub

O Git é um Sistema de Controle de Versão Distribuído que evita que alterações realizadas em um projeto modifiquem o código-fonte.

Sql (PostgreSQL)

Bancos de dados muito populares, como Oracle Database, MySQL, PostgreSQL e Microsoft SQL Server são alguns adeptos do SQL.

MongoDB

O MongoDB é um banco de dados orientado a documentos que possui código aberto (open source) e foi projetado para armazenar uma grande escala de dados.
